feature
该板子主要面向开发
支持使用STM32F103C8的所有外设，以及支持所有引脚外用
兼容STM32F030C8
主要用于项目打板前的方案验证以及驱动编写或可直接做控制核心使用

resourse
1.该板子板载两个LED，可分别通过跳线帽连接至PB2-PA15或使用杜邦线连接至其它IO
2.板载3个user按键，未加电容进行硬件消抖以便引脚另作他用。一个按键接至PA4，另外两个两个按键分别接至PA0（WAKE_UP引脚）和PC13（TAMPER_RTC引脚）以便使用睡眠唤醒功能以及RTC中断功能
3.一个Typec接口经过跳线帽连接至PA11-PA12以使用USB外设，移除跳线帽PA11-PA12可用于其它功能
4.板载31.768K晶振以使用RTC外设
5.板载电池接至BAT引脚以使用后备寄存器进行掉电保存
6.板载兼容7脚OLED以及8脚TFTLCD接口，可使用SPI外设驱动
7.输出接口全部做丝印标识，电源接口分开，下载接口和调试串口单独引出
